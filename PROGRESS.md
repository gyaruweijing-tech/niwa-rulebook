# 丹羽ルールブック — PROGRESS

「これは」と思った価値観を保存して育てる、自分ルール帳PWA。

## コンセプト

- **三箇条**（最大3つ・人生の憲法）＋ **心得**（上限なし・候補層）の二層構造
- 心得 → 三箇条へ**昇格**。三箇条が満杯なら誰かを**卒業**（削除ではなくアーカイブ、理由も残す）させないと昇格できない
- **今日の1本**: 心得から日付シードのランダムで1件表示（AI不使用・毎日変わる・同じ日は同じ）
- データは端末のlocalStorage（サーバーレス）。Markdown / JSON で**お引越し**（エクスポート/インポート）できる

## 技術スタック

- Vite + React + TypeScript
- vite-plugin-pwa（オフライン対応・ホーム画面追加可）
- 配布: GitHub Pages（`.github/workflows/deploy.yml` で main へのpush時に自動デプロイ）
- Vercel不使用・サーバーコードなし・課金ゼロ
